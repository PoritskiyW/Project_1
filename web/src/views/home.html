Description about project

Командой разработчиков "Джаваскриптизеры" был создан вводный проект для ознакомления с Front-End направлением в сфере IT. Перед командой стояла задача - разработать проект прилагая к этому максимум усилий, при этом было запрещено использовать библиотеки. Функционалом проекта является возможность записывать, редактировать а также удалять информацию с дальнейшим сохранением данных на стороне пользователя в поддерживаемые форматы, такие как:
CSV JSON YAML XML
